# 🐶🐱 Cat vs Dog Image Classifier

Bu proje, kedileri ve köpekleri ayırt edebilen basit bir CNN tabanlı görüntü sınıflandırıcıdır. Model, TensorFlow ve OpenCV kullanılarak eğitildi.

## 🚀 Projeyi Çalıştırmak

```bash
git clone https://github.com/kullaniciadi/cat_dog_classifier.git
cd cat_dog_classifier
pip install -r requirements.txt
```

## 🧠 Model Eğitimi

```bash
python src/model.py
```

## 📷 Tahmin Yapmak

```bash
python src/detect.py
```

## 📁 Veri Yapısı

```
data/
    train/
        cat.1.jpg
        dog.2.jpg
    test1/
        5940.jpg
```
